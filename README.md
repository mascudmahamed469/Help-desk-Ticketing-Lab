Help Desk Ticketing System Lab
Overview
Simulated IT help desk environment built using Spiceworks Cloud Help Desk to practice real world IT support workflows.
Tools Used
Spiceworks Cloud Help Desk
What I Built
Deployed and configured a cloud based ticketing system and resolved 5 simulated support tickets covering the most common help desk scenarios.
Tickets Resolved
Ticket 1 — Welcome and Orientation
Reviewed system orientation and familiarized with dashboard, filters, split view, and ticket management interface.
Ticket 2 — System Import Assessment
Evaluated ticket import process and determined not applicable for new deployment. Documented reasoning and closed appropriately.
Ticket 3 — Account Lockout
Simulated Windows account lockout scenario. Practiced identity verification, Active Directory unlock procedure, password reset with forced change on next login, and user confirmation.
Ticket 4 — WiFi Connectivity Issue
Diagnosed and resolved network connectivity failure using systematic troubleshooting including basic checks, network isolation, Windows troubleshooter, and command line network stack reset via netsh winsock reset and ipconfig commands.
Ticket 5 — Printer Offline
Resolved printer offline issue by clearing stuck print queue and restarting Print Spooler service via services.msc. Documented root cause and provided user with preventive guidance.
